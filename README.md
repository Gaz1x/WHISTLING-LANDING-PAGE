# Для билда проекта использовать
```
npm run build
```

# После команды
#### Всё содержимое папки /build перенести в корневую папку 

# Важно
#### Файлы билда страницы хранить в отдельной ветке gh-pages

# Используемые библиотеки
1. ChakraUI - компоненты ``` npm i @chakra-ui/react @emotion/react @emotion/styled ```
2. Web Audio Api - работа с микрофоном
3. Lucide - иконки ```npm install lucide-react```
