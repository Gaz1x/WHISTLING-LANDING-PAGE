# Для билда проекта использовать
    npm run build

# После команды
    всё содержимое папки /build перенести в корневую папку 

# Важно
    Файлы билда страницы хранить в отдельной ветке gh-pages

# Используемые библиотеки
    ChakraUI - компоненты ( npm i @chakra-ui/react @emotion/react @emotion/styled )
    Framer Motion - анимации ( npm install framer-motion )
    Web Audio Api - работа с микрофоном
